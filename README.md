# iOS_UHF_Sample_Swift
* UHF Sample App using Swift
* Only demo the following functions:
> * BLE scanning
> * BLE connecting
> * Get firmware version command

## Supported Devices
* TS100
* TS100A
* TS800-00
* UR0250
* TS800-60

## Pre-requisites
* iOS 14.0+
* Xcode 14.1
* Sample v1.0.0.1
* SDK v3.1.5.2

## Installation
* Please download this project and open with Xcode.
